Privacy Policy

Last updated: 2026

1. Introduction

This real-time screen translation application respects user privacy and is committed to protecting user data.

This application operates locally on the device and does not collect personal data.

2. Data Collection

The application:

  • Does NOT collect personal data
  • Does NOT store screen captures
  • Does NOT transmit screen content to external servers
  • Does NOT share data with third parties

All processing is performed locally on the user's device.

3. Screen Capture (MediaProjection API)

The application uses Android's official MediaProjection API to:

  • Temporarily capture on-screen content
  • Detect text using OCR
  • Translate detected text

Important:

  • Captured images are processed in memory only
  • No screen content is stored
  • No screen content is transmitted
  • Data is immediately discarded after processing

The screen capture feature is activated only after explicit user permission.

4. Overlay Permission

The application uses the "Display over other apps" permission solely to:

  • Show a floating translation button
  • Allow users to select a screen area for translation

The overlay does not monitor, record, or intercept user interactions.

5. Text Processing

Detected text is processed locally.

If online translation services are used, only the extracted text is transmitted for translation — not the full screen image.

No personal identifiers are attached to translation requests.

6. Permissions Used

The app may request the following permissions:

  • Display over other apps (Overlay)
  • Screen capture permission (MediaProjection)
  • Internet access (only if online translation is enabled)

These permissions are strictly required for the core functionality of the app.

7. Advertisements (If Applicable)

If advertisements are integrated in the future:

  • Third-party ad providers may use anonymous advertising identifiers
  • The application itself does not collect personal user data

8. Data Security

All data processing occurs locally.

No external database or user account system is used.

9. Children's Privacy

This application does not knowingly collect personal data from children.

10. Changes to This Policy

This Privacy Policy may be updated periodically. Updates will be published on this page.
