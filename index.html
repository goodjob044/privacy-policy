Privacy Policy

Last updated: May 1, 2026

1. Introduction

LiveLens is a real-time screen translation application. This policy explains how the application handles data and what information, if any, is transmitted externally. Privacy is a core principle of this application — no personal data is collected, stored, or sold.

2. Summary of Data Practices

  • Does not collect personal data
  • Does not store screen captures beyond immediate processing
  • Does not transmit screen images or extracted text to external servers
  • Does not share user data with third parties
  • Uses an internet connection only to download translation language models from Google's servers — no user content is involved in this process

3. Screen Capture (MediaProjection API)

LiveLens uses Android's official MediaProjection API to capture on-screen content for the purpose of text detection and translation. The following applies to all screen captures:

  • Captured images are temporarily written to the application's private cache directory for inter-process communication between the capture service and the main application
  • These files are deleted immediately after processing is complete
  • Cache files are never accessible to other applications
  • Captured content is never transmitted to any external server
Screen capture functionality is activated only after explicit user permission is granted via Android's standard permission dialog.

4. Overlay Permission (Display Over Other Apps)

LiveLens requests the "Display over other apps" permission exclusively to:

  • Display a floating translation button on screen
  • Allow users to select a screen region for translation
  • Show translation results in an overlay interface

The overlay does not monitor, record, or intercept any user interactions outside of the application's own interface elements.

5. Camera

The camera permission is used solely to allow users to take photos of physical objects (such as documents, signs, or menus) for text extraction and translation. Photos taken through LiveLens are:

  • Processed locally on the device
  • Never stored permanently by the application
  • Never transmitted to external servers

6. Text Recognition (OCR)

Text recognition is performed entirely on-device using Google ML Kit's Text Recognition engine. This engine uses models bundled within Google Play Services and does not require an internet connection to function.

  • No screen images are uploaded for recognition
  • No extracted text is sent to external servers during OCR
  • All processing happens locally in real time

7. Translation

Translation is performed on-device using Google ML Kit's Translation API. Once a language model is downloaded, all translation processing occurs locally — no text is transmitted externally for translation.

Language model downloads: Before a language pair can be used for the first time, the corresponding translation model must be downloaded from Google's servers. During this download:

  • Only the model file is downloaded — no user content is transmitted
  • The download occurs over an encrypted (HTTPS) connection
  • Downloaded models are stored locally on the device for future offline use
Language model downloads are subject to Google's Privacy Policy:
https://policies.google.com/privacy

8. Third-Party SDKs

LiveLens integrates the following Google SDK:

  • Google ML Kit (Text Recognition + Translation) — used for on-device OCR and translation. Model downloads are handled by Google's infrastructure. No user data is shared with Google through these SDKs during normal application use.
    ML Kit Terms of Service

No advertising SDKs or analytics SDKs are integrated in this application.

9. Permissions Used

  • Display over other apps (SYSTEM_ALERT_WINDOW) — required to display the floating overlay interface
  • Screen capture (MediaProjection) — required to capture screen content for text extraction
  • Camera (CAMERA) — required to take photos of physical text for translation
  • Internet (INTERNET) — required solely to download translation language models from Google ML Kit servers. Used one time per language pair; not required after models are downloaded.
  • Foreground Service (FOREGROUND_SERVICE, FOREGROUND_SERVICE_MEDIA_PROJECTION, FOREGROUND_SERVICE_SPECIAL_USE) — required to keep the capture and overlay services running while the user is using other applications

10. Data Security

All sensitive data processing occurs locally on the device. Temporary files created during screen capture are stored in the application's private cache directory and are deleted immediately after use. No external database, user account, or cloud storage is used.

11. Children's Privacy

LiveLens does not knowingly collect personal data from children under the age of 13, or any other age threshold defined by applicable local law. The application does not collect personal data from any user regardless of age.

12. Changes to This Policy

This Privacy Policy may be updated periodically to reflect changes in the application or applicable regulations. The "Last updated" date at the top of this page will be revised accordingly. Continued use of the application after any update constitutes acceptance of the revised policy.
